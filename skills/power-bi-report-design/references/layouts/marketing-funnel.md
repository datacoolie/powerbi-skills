## Theme + iconography guidance

- **Channel palette**: stable mapping (Paid Search = blue, Organic = green, Social = orange, etc.) вҖ” apply in theme `dataColors` so donut/funnel/table agree
- **Logo:** company wordmark top-left of title bar at `(32, 24)`, max height 28px. Do NOT use channel partner logos inside the donut вҖ” show their name in the legend only (avoid trademark misuse).
- **Icons on KPIs**: funnel, dollar, megaphone, arrow-right
- Use data bars (not heat background) on attribution revenue column

## When NOT to use this layout

- вқҢ Brand-awareness only campaigns (no conversion funnel yet) вҖ” use trend + reach card
- вқҢ Single-channel deep dive вҖ” use `sales-performance` with channel filter instead
- вқҢ Creative asset library вҖ” not a reporting page (use SharePoint)
