## Theme + iconography guidance

- **Palette:** neutrals + green/amber/red traffic-light triad only вҖ” no other accents
- **Logo:** company wordmark top-left of title bar at `(32, 24)`, max height 28px вҖ” scorecards are formal exec docs, logo is expected. Period toggle pins to top-right of same bar; do not compete with logo.
- **Icons:** one per perspective (dollar, users, gear); place top-right of each tile
- **Status strip**: 6px left-edge colored bar is the primary signal, not the text

## When NOT to use this layout

- вқҢ Need to explain movements вҖ” no room for narrative; pair with exec-overview
- вқҢ KPIs require different time-grain per tile (scorecard assumes shared period toggle)
- вқҢ Fewer than ~8 KPIs вҖ” the grid looks sparse; use `exec-overview-16x9` instead
